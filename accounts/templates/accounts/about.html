About

Connect People is a simple one-to-one Chat Application. Where an user can send or receive text messages. Also an user can see onther user is online or not and if not then lase seen time.


There is a simple registaraton form to open an account. Anyone can do register and do chat with other users. It is built on the basis of an organizational internal communication. This website has been showcased for a demo.
